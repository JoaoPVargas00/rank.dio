Desafio: Calculadora de Partidas Rankeadas
O objetivo aqui foi aplicar funções para isolar a lógica de cálculo.

Regras de Negócio:
Cálculo: Vitórias - Derrotas

Níveis: Variam de Ferro (até 10 vitórias) a Imortal (acima de 101 vitórias).

Tecnologias Aplicadas:
Modularização com Funções.

Retorno de valores compostos.

Operadores aritméticos e lógicos.
